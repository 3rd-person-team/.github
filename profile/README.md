**3인칭 관찰자 시점**은 AI를 활용하여 빠르게 게임을 만드는 개발팀입니다.

현재 **로그라이크 디펜스 게임**을 개발하고 있습니다.  
이곳에는 프로젝트 코드와 개발 도구, 관련 문서를 정리합니다.
